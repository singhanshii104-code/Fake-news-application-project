# Fake News Detection Application

This is my first web development project! It uses **AI** to help identify whether a piece of news is real or fake.

# Features
*   **Frontend**: Built with HTML and CSS for a clean user interface.
*   **Backend**: Powered by Node.js (`server.js`).
*   **AI Integration**: Uses the OpenAI API to analyze text credibility.

# How to Setup
1.  Clone this repository.
2.  Run `npm install` to get the necessary dependencies.
3.  Create a `.env` file and add your `OPENAI_API_KEY`.
4.  Run `node server.js` to start the application.
